# How to Download the APK

Follow these steps to download and install the app APK file.

---

## ðŸ”¹ Step 1: Go to the Releases Page

1. Open this repository on GitHub.
2. Click on **Releases** (on the right side of the page).
   
Or go directly to: [Release Page](https://github.com/1-noob/RageBait/releases/tag/APK)


---

## ðŸ”¹ Step 2: Download the APK

1. Under the latest release, find the **Assets** section.
2. Click on the file that ends with: `.apk`


3. The download will start automatically.

> Do not download:
> - `Source code (zip)`
> - `Source code (tar.gz)`
>
> Those are the project source files, not the installable app.

---

## ðŸ”¹ Step 3: Install the APK (Android)

1. Open the downloaded `.apk` file on your Android device.
2. If prompted, allow installation from unknown sources.
3. Tap **Install**.
4. Open the app once installation is complete.
